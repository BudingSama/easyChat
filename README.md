# react-easychat
- **version**
```
verison 1.1.9
```

- **quick start**

```
$ npm install react-easychat -S
```
- **options**

```
  const options = {
    nickname:'',//可选 助手名称
    defaultTalk:[],//可选 默认发送信息列表 maxlengh <= 3
    status:'',//可选 助手在线状态 在线/离线/忙碌/离开
    chatStatus:'',//可选 助手聊天状态
    placeholder:'',//可选 输入框 placeholder
  }
      
      
  <EasyChat options={options} />
```
